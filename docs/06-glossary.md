# Glossary

- AR = Accounts Receivable
- AP = Accounts Payable
- VAT = Value Added Tax
- GL = General Ledger
- TB = Trial Balance
- FS = Financial Statement
- PP30 = VAT monthly filing
- PND1 = Payroll withholding tax
- PND3 = Withholding tax for individuals
- PND53 = Withholding tax for juristic persons
- SSO = Social Security Office
- Reconciliation = Matching accounting records with external or subledger records
- Closing Period = Locking an accounting period after review

## ประเภทธุรกิจ (Business Type)
ข้อมูลที่ใช้ระบุลักษณะการประกอบกิจการของบริษัท เพื่อใช้กำหนดรูปแบบข้อมูล เอกสารบัญชี รายงาน
วิธีตรวจสอบ และเงื่อนไขทางบัญชี/ภาษีที่เกี่ยวข้องกับระบบ — ระบบต้องรองรับลูกค้าทุกประเภทธุรกิจ
โดยไม่ hard-code สมมติฐานเฉพาะอุตสาหกรรมใด

| ประเภทธุรกิจ | ความหมาย |
|---|---|
| ซื้อมา-ขายไป / Trading | ซื้อสินค้ามาขายต่อ ไม่ได้ผลิตเอง |
| ผลิต / Manufacturing | ซื้อวัตถุดิบมาแปรรูปหรือผลิตเป็นสินค้า |
| บริการ / Service | รายได้หลักมาจากค่าบริการ |
| รับเหมา / Contractor | รับงานเป็นโครงการ มีงวดงาน/ต้นทุนโครงการ |
| นำเข้า-ส่งออก / Import-Export | มีธุรกรรมต่างประเทศ ภาษีนำเข้า อัตราแลกเปลี่ยน |
| ขนส่ง / Logistics | รายได้จากค่าขนส่ง ค่าระวาง ค่าบริการจัดส่ง |
| อสังหาริมทรัพย์ / Real Estate | ซื้อขาย/ให้เช่า/พัฒนาอสังหาฯ |
| โรงงานผลิต | เน้นวัตถุดิบ งานระหว่างทำ สินค้าสำเร็จรูป ต้นทุนผลิต |
